# Blog Application

This is a simple blog application built with Go and MongoDB. It provides a RESTful API to manage blog posts.

## Features

- Get a list of posts
- Get a single post
- Create a new post
- Update an existing post
- Delete an existing post

## Getting Started

### Prerequisites

- Go 1.18+
- MongoDB (local or remote)
- Git
